Validator for Anomaly Detection in Time Series

{# Vadetis is web application to perform, compare and validate various anomaly detection algorithms using different configurations. It allows users to upload their own datasets#} {# as well as training data in order to perform outlier detection. The datasets can either be shared with other users or only be used by yourself.#} {# The datasets can be altered by injecting additional outliers.#} {# Look into the FAQ page for more information about the usage of this application.#} {{ Title }} is an online tool to Visualize The Repair of Anomaly Repair Techniques.

{#
Features
#}

Main Parts:

  • Display
    • Datasets
  • Repair
    • Repair Synthetic Datasets
    • Inject Data and Repair
  • Algorithm Analysis
    • Detailed Repair of Matrix decomposition Techniques
    • Display Parameter Search using Bayesian Optimization
  • Recommendation
    • Real-World Datasets
    • Synthetic Datasets

Supported algorithms:

  • Iterative Minimum Repairing (IMR) [VLDB 2017]
  • Stream Data Cleaning under Speed Constraints (SCREEN) [SIGMOD 2015]
  • CDrep (Repair based on the Centroid Decomposition)
  • RPCA (Robust PCA using Huber Loss Function)

This application was implemented at the at the eXascale Infolab, a research group at the University of Fribourg, Switzerland.
